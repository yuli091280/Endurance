# stuff

## setup

On Windows, do these:

- find and install the qt6 runtime library

On Linux, do these:

- `sudo apt update && sudo apt install qt6-base-dev`

then do these for python stuff:

- `pip install poetry`
- `poetry install`
- `poetry shell`

## Running the project

do this: `pyqt-test`

## building the project

`make clean build`
